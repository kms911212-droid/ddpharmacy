繁體中文 🇹🇼 🇭🇰 🇲🇴

產品摘要
  • Acnon Cream適合發炎性痘痘的局部護理。
  • 特別適合紅腫痘痘、疼痛痘痘與有膿的痘痘。
1️⃣ 適用痘痘類型
  • 紅腫痘痘
  • 疼痛的發炎性青春痘
  • 有膿的膿皰型痘痘
  • 除了臉部，也可用於背部、胸前痘痘。
2️⃣ 主要成分
  • Ibuprofen Piconol：幫助舒緩發炎
  • Isopropylmethylphenol：幫助抑制痘痘細菌
  • 屬於抗發炎 + 抗菌的痘痘護理配方。
3️⃣ 使用方法
  • 洗臉後擦乾水分再使用。
  • 僅薄擦於痘痘部位。
  • 一天可使用數次。
  • 也可在基礎保養吸收後再局部塗抹。
4️⃣ 注意事項
  • 請勿使用於眼睛、眼周、深層傷口或燙傷部位。
  • 避免大範圍過量塗抹。
  • 若出現紅疹、搔癢、乾燥、灼熱感或腫脹，請停止使用並諮詢專業人員。
5️⃣ 與Aclean Gel的差異
  • Acnon Cream：較適合紅腫、發炎型痘痘
  • Aclean Gel：較適合粉刺、黑頭、白頭等非發炎性痘痘
  • 若一起使用，建議順序為 洗臉 → Aclean Gel → 基礎保養 → Acnon Cream。

  • 赤ニキビ

  • Isopropylmethylphenol：アクネ菌対策をサポート
  • 抗炎症 + 抗菌のWアプローチの処方です。
3️⃣ 使い方
  • 洗顔後、水分をよく拭き取ってから使ってください。
  • ニキビ部分にのみ薄く塗ってください。
  • 1日に数回使用できます。
  • スキンケア後、なじんでから部分使いすることもできます。

English 🇺🇸

Product Summary
  • Acnon Cream is a spot treatment cream for inflammatory acne.
  • It is suitable for red, painful, and pus-filled pimples.
1️⃣ What type of acne is it for?
  • Red inflamed pimples
  • Painful inflammatory acne
  • Pus-filled acne
  • It can be used not only on the face but also on acne on the back and chest.
2️⃣ Key Ingredients
  • Ibuprofen Piconol: helps reduce inflammation
  • Isopropylmethylphenol: helps control acne-causing bacteria
  • This formula provides both anti-inflammatory and antibacterial care.
3️⃣ How to Use
  • Use after cleansing and drying the skin.
  • Apply a thin layer only to the acne area.
  • It may be used several times a day.
  • It can also be applied after basic skincare has fully absorbed.
4️⃣ Caution
  • Do not use on the eyes, around the eyes, deep wounds, or burn areas.
  • Avoid applying too much over a large area.
  • Stop use and ask a pharmacist or healthcare professional if severe rash, itching, dryness, burning, or swelling occurs.
5️⃣ Difference from Aclean Gel
  • Acnon Cream: better for red, inflamed acne
  • Aclean Gel: better for whiteheads, blackheads, and other non-inflammatory acne
  • If using both, the usual order is cleanser → Aclean Gel → basic skincare → Acnon Cream.
